📦 Ollama のセットアップ方法
  1. Ollama をインストール: https://ollama.com からダウンロード
  2. モデルをダウンロード（ターミナルで）:
    ollama pull llama3.2　または　ollama pull qwen2.5
  3. Ollama を起動: メニューバーのアイコンから起動、または ollama serve
  4. 上の「接続テスト」ボタンで確認 → 次へ進む

翻訳精度が高いおすすめモデル: qwen2.5:7b（中国語に強い）、llama3.2:3b（軽量・高速）、gemma2:9b（多言語）

⚡ 速度について

ローカルLLMはセルごとにモデルを呼び出すため、Google翻訳より時間がかかります。
目安: llama3.2:3b で 1セル約2〜5秒 / qwen2.5:7b で 1セル約5〜10秒。
行数が多い場合は軽量モデルを推奨します。
